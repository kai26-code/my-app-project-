## App Description
The Time Budget Planner helps users manage their weekly time more effectively by treating it like a financial budget. The app allows users to input the number of hours spent each day of the week, set a weekly time goal, and allocate time from total hours across categories such as study, work, and personal activities. It then analyzes the data using accounting and finance concepts including variance analysis, efficiency measurement, and time valuation which provide insights into how time is being used by the student and/or accountant. In addition, the app creates visualizations and smart recommendations to help users improve productivity, reduce inefficiencies, and make better decisions about how they allocate their time throughout the week. 
 
## Intended Users
This app is intended for both students and professionals who want to manage their time more effectively. For students, it helps track study hours, plan assignments, and optimize productivity. For accounting and finance professionals, the app can be used to allocate work hours across tasks, analyze efficiency, and make informed decisions similar to how financial budgets are monitored and evaluated. Essentially, it is meant for anyone who wants to treat their time as a valuable resource and make data-driven decisions to improve performance.
   
## Final Features 
1. Daily time tracking
2. Goal comparison & variance analysis
3. Visualizations
4. Efficiency analysis (productive vs total hours)  
5. Time value estimation using hourly rate  
6. Break-even analysis for goal tracking  
7. Category-based time allocation (Study, Work, Personal)  
8. Dynamic workload-based recommendations  
9. Real-time conditional feedback messages  
10. Interactive user inputs (radio, slider, selectbox)

## How to use the app 
1. Enter the number of hours you spend each day of the week  
2. Set your weekly goal (total hours you want to achieve)  
3. Select your main focus (Study, Work, or Personal)  
4. Category Allocation: Breakdown your total hours into Study, Work, & Personal hours with amount of hours you are more productive
5. Select the percentage you anticipate time value or output to increase if productivity habits improve.
6. If you are a student, estimate how much could earn for each hour studying. For Accountants, how much money would earn each hour working.
7. Choose workload level: Based on amount of hours, is the workload simple (low) for you? is it busy (moderate)? or is it too much (high)?  
8. Click **Run Analysis** & view results automatically.
9. Review your dashboard: 
   - Total hours
   - Variance vs goal
   - Efficiency score
   - Budget vs Actual Analysis & Break Even
   - Recommendations
   - Visual charts
